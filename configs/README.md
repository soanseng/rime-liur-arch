# 蝦米輸入法配置版本說明

本文件夾包含兩個版本的蝦米輸入法配置文件，使用者可以根據需求選擇合適的版本。

## 版本差異

### 1. 基礎版（中打不含英文詞庫）（liur.chinese-only.schema.yaml）
**精簡版本，專注於中文輸入體驗**

**功能特色：**
- ✅ 完整的蝦米中文輸入
- ✅ 英文仍可輸入，但需手動輸入完整單詞後按 Enter 上屏
- ✅ 中英文混合自定詞功能
- ✅ 符號清單、變體英數、日期時間等擴充功能
- ✅ 拼音、注音輸入支援
- ❌ 不含英文詞庫（easy_en）
- ❌ 移除英文大小寫轉換功能

**適合使用者：**

- 主要進行中文寫作的使用者
- 希望減少英文候選干擾的使用者
- 追求輸入法精簡性能的使用者
- 純中文寫作、文學創作者

### 2. 完整版（中打含英文詞庫版）（liur.schema.yaml）
**完整功能版本，中文輸入搭配英文詞庫輔助**

**功能特色：**
- ✅ 完整的蝦米中文輸入
- ✅ 英文詞庫支援（easy_en.dict.yaml）
  - 英文單詞會出現在候選選單中
  - 支援英文自動補全（如：app → apple）
  - 英文可按空白鍵選擇上屏
- ✅ 英文大小寫轉換（word] → Word，word]] → WORD）
- ✅ 中英文混合自定詞功能
- ✅ 符號清單、變體英數、日期時間等擴充功能
- ✅ 拼音、注音輸入支援

**適合使用者：**
- 經常需要輸入英文單詞的使用者
- 希望英文有候選提示和自動補全的使用者
- 需要英文大小寫快速轉換功能的使用者
- 程式開發、技術文檔撰寫者

## 版本選擇建議

| 使用情境 | 建議版本 |
|---------|---------|
| 純中文寫作、文學創作 | 基礎版（中打不含英文詞庫） |
| 學術論文（中文為主） | 基礎版（中打不含英文詞庫） |
| 希望減少英文候選干擾 | 基礎版（中打不含英文詞庫） |
| 日常中英文混合輸入 | 完整版（中打含英文詞庫版） |
| 程式開發、技術文檔 | 完整版（中打含英文詞庫版） |
| 商務文件（需要英文） | 完整版（中打含英文詞庫版） |

## 安裝方式

### 自動安裝（推薦）
使用安裝腳本會自動提示選擇版本：

**Windows：**
```powershell
.\rime_liur_installer.ps1
```

**macOS/Linux：**
```bash
./rime_liur_installer.sh
```

### 手動安裝
1. 選擇合適的配置文件
2. 複製到根目錄並重新命名為 `liur.schema.yaml`

**選擇基礎版（中打不含英文詞庫）：**
從 `configs` 資料夾複製 `liur.chinese-only.schema.yaml` 到 Rime 使用者資料夾，並重新命名為 `liur.schema.yaml`

**選擇完整版（中打含英文詞庫版）：**
從 `configs` 資料夾複製 `liur.schema.yaml` 到 Rime 使用者資料夾

## 常見問題

**Q: 基礎版還能輸入英文嗎？**
A: 可以，但沒有英文詞庫提示，需要手動輸入完整單詞後按 Enter 上屏。

**Q: 自定詞功能在兩個版本都有嗎？**
A: 是的，自定詞功能對使用者很重要，兩版本都保留。
